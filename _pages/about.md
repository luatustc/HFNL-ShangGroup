本网站用于整理 ShangGroup 两节点 CPU SLURM 集群的使用说明，内容包括：

- 集群总体硬件配置；
- CPU 节点、存储与网络说明；
- 登录方式与账号申请流程；
- Slurm 作业提交方法；
- 常用软件环境与使用建议；
- 常见问题与注意事项。

## 集群定位

该集群为纯 CPU 计算集群，主要面向量子化学计算、AFQMC、数值模拟、代码编译测试、科研数据处理等任务。当前集群不含 GPU 节点，不适合直接承接 CUDA/GPU 深度学习训练任务。

## 更新原则

服务器硬件、系统版本、队列限制和软件环境可能随维护情况变化。若页面内容与实际服务器不一致，请以管理员通知和服务器实时命令输出为准，并及时更新本文档。

## 安全说明

本网站如果部署在 GitHub Pages 上，默认可能被公开访问。请不要在页面中写入真实密码、root 信息、公网 IP、跳板机细节、内网监控地址、管理员手机号等敏感信息。
