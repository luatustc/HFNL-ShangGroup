## 4. 使用方法

示例脚本 `cpu.sh`：

```bash
#!/bin/bash
#SBATCH -J test_cpu
#SBATCH -p normal
#SBATCH -N 1
#SBATCH -n 64
#SBATCH -o %j.log
#SBATCH -e %j.err

echo "SLURM_JOB_PARTITION=$SLURM_JOB_PARTITION"
echo "SLURM_JOB_NODELIST=$SLURM_JOB_NODELIST"
echo "Time is $(date)"
echo "Directory is $PWD"

module purge
module av

# 在这里写你的程序命令
srun hostname
