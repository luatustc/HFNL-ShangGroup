## 0. HPC 集群介绍

本页面用于介绍323实验室计算服务器的基本情况、登录方式、队列设置和使用注意事项。

实验室服务器主要包括：

| 类型 | 数量 | 用途 |
|---|---:|---|
| 登录节点 | 1 | 用户登录、代码编辑、任务提交 |
| CPU 计算节点 | 若干 | 大规模 CPU 并行计算 |
| GPU 计算节点 | 若干 | 深度学习、量子化学、神经网络训练 |
| 存储节点 | 若干 | 数据存储、结果备份 |

## 1. 适用对象

本服务器主要面向实验室内部成员使用，包括：

1. 在读硕士生；
2. 在读博士生；
3. 博士后；
4. 经导师或管理员批准的合作者。

## 2. 使用流程

1. 向管理员申请账号；
2. 获取登录方式；
3. 使用 SSH 登录；
4. 编写 Slurm 脚本；
5. 使用 `sbatch` 提交任务；
6. 使用 `squeue` 查看任务状态；
7. 计算结束后整理结果并及时备份。

## 3. 注意事项

- 不要在登录节点长时间运行大型计算任务；
- 大型任务必须通过 Slurm 提交；
- 不要公开服务器密码、内网 IP、跳板机地址等敏感信息；
- 重要数据应定期备份；
- 如任务异常占用大量资源，应及时联系管理员。
